# T11 — 数值平衡（生存循环）

## 目标
让"饥饿/口渴"成为真实但可控的压力，形成「捕食→进食→成长」的正循环；同时确保刚开局脆弱的幼体有喘息空间。

## 现状问题
- `HUNGER_INTERVAL=35s` / `THIRST_INTERVAL=30s`：每点衰减需 30~35 秒，满值撑近 1 小时，生存压力几乎为零，进食循环无存在感。
- 尸体 `LIFETIME=20s` / `MAX_BITES=3`：击杀后 20 秒就腐烂，脆弱幼体常在战斗中被打断、来不及吃。

## 方案（数值为手感假设，待真机调）
1. **PlayerDino.gd**
   - `HUNGER_INTERVAL` 35.0 → **14.0**（满→空约 23 分钟；5 分钟会话掉 ~21 点，有饥饿感）。
   - `THIRST_INTERVAL` 30.0 → **12.0**（静止满→空约 20 分钟；冲刺时倍率 0.5 → 间隔减半，鼓励节制狂奔）。
   - 保留 `STARVE_DAMAGE=6` / `THIRST_STARVE_DAMAGE=5` / `THIRST_SPRINT_MULT=0.5`。
2. **AICorpse.gd**
   - `LIFETIME` 20.0 → **35.0**（击杀后留更多时间进食）。
   - `MAX_BITES` 3 → **4**（单具尸体可吃 4 口，肉食龙成长更顺）。

## 效果预期
- 一次会话中玩家需进食 1~2 次、饮水数次，循环有存在感但不压迫。
- 幼体击杀猎物后能在尸体腐烂前吃完。
- 未改动战斗/生态 aggression，避免破坏 T6 已平衡的难度。
- 零 SCRIPT ERROR / Animation not found。
